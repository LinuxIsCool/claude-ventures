# /ventures Command

Show the current state of your venture portfolio, sorted by priority.

## Instructions

1. Call the `venture_portfolio` MCP tool to get the full portfolio dashboard
2. Display the results clearly formatted
3. Highlight any deadlines within 14 days
4. Note any dormant ventures with active connections

## Quick Actions

After viewing, you can:
- Create a new venture: `venture_create`
- View venture details: `venture_get`
- See timeline: `venture_timeline`
- Check financials: `venture_financials`
- Transition a venture: `venture_transition`
