# Portfolio Analysis

## When to Use
- Start of day/session orientation
- Deciding where to focus attention
- Reviewing overall venture health
- Making strategic allocation decisions

## Approach

1. **Get the big picture**: `venture_portfolio({})` for full dashboard
2. **Check urgency**: `venture_timeline({ include_overdue: true })` for deadline pressure
3. **Analyze allocation**: Compare time spent vs priority ranking
4. **Identify gaps**: Ventures with high priority but low recent activity
5. **Surface connections**: Dormant ventures that connect to current active work

## Attention Allocation Framework

Priority determines attention order, but not time allocation:
- **Cliff deadlines** (external, < 14 days): Block dedicated time
- **High priority active**: Daily touchpoints
- **Exploring ventures**: Dedicated exploration sessions
- **Sustaining ventures**: Periodic check-ins
- **Dormant ventures**: Only when context makes them relevant

## Health Indicators

- **Healthy**: Active ventures have recent updates, deadlines tracked, milestones progressing
- **Attention needed**: High-priority venture with no updates in > 7 days
- **Risk**: External deadline < 14 days with incomplete milestones
- **Opportunity**: Dormant venture connected to current active work
